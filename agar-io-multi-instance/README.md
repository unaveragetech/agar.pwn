# Agar.io Multi-Instance Controller

This project allows you to control multiple **Agar.io** instances on **one screen** using **one set of keyboard and mouse inputs**.

## Features
- Control multiple **Agar.io** instances from a single browser window.
- Sync mouse and keyboard inputs across all instances.
